## 📌 Tips visuales rápidos

- 🟥 **Evita rojo puro (#FF0000)**: grita como jefe enojado.
- ✅ **Usa azul/gris para elegancia:** calma y legibilidad.
- 📈 **Siempre pon títulos a tus gráficas. Siempre.**
- 🎨 **Máximo 5 colores por gráfico. No es piñata.**
- 🔢 Formatea números:
  - `f"{x:,.0f}"` → `1,000`
  - `f"{x:.2%}"` → `32.57%`
